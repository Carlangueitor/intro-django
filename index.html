1.2 Instalación de Django

Windows:

          
            # Descargar e instalar Pip: 
            # http://www.lfd.uci.edu/~gohlke/pythonlibs/#pip
            $ pip install django

Unix (Linux/Mac):

          
            $ easy_install pip
            $ pip install django

Verificamos

          $ python -c "import django; print(django.get_version())"
